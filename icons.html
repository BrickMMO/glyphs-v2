Frequently asked questions

Got questions? Here are answers to the most common ones from the BrickMMO community.

Are the icons free to use?

Yes. All icons in Glyphs v2 are free to use for personal and community projects within BrickMMO. For commercial projects, please check the license details on our docs page.

How do I use the icons in my project?

Simply include the BrickMMO Glyphs CSS file and use the appropriate icon classes. For example: <i class="icon icon-dashboard"></i>. Check our documentation for complete usage instructions.

Can I customize the icon styles?

Absolutely! You can customize the icons using CSS. Change the font-size, color, and other properties to match your design. See our customization guide in the documentation.

Are new icons added regularly?

Yes! We regularly add new icons based on community feedback and BrickMMO project needs. Follow our updates on GitHub or check back here for the latest additions.
